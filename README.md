# MVx Patterns for Unity Windows

A collection of my personal implementations of different **MVx architectural patterns** for window logic in Unity projects.

This repository is created as an architecture showcase.
Each branch contains a separate implementation of a specific approach, such as **MVP**, **MVC**, **MVVM**, or other MVx-style variations.

## Available Branches

| Branch | Pattern              | Description                                     |
| ------ | -------------------- | ----------------------------------------------- |
| `main` | —                    | General repository overview                     |
| `MVP`  | Model-View-Presenter | MVP-based implementation for Unity window logic |

More branches may be added later as new architectural approaches are implemented.
